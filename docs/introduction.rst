Introduction
============
Tellurium is a cross-platform and open source Python based environment that integrates a variety of useful packages for systems biology modeling. The IDE is based on the [spyder2](https://github.com/spyder-ide/spyder)

|More information is available from the project homepage on  
|http://tellurium.analogmachine.org/
